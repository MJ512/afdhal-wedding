# Image Asset Guide

Drop final production artwork into this folder using these exact names:

- `lantern1.png`: realistic transparent PNG lantern, warm gold, main foreground depth.
- `lantern2.png`: second realistic transparent PNG lantern, slightly smaller or different silhouette.
- `mosque-bg.png`: subtle transparent Madina-style mosque skyline or silhouette for the hero background.
- `afdhal-card-reference.jpeg`: the original invitation card reference, kept for design direction.

Recommended export notes:

- Lantern PNGs should be at least 600 px tall with transparent backgrounds.
- Mosque artwork should be wide, soft-edged, and low contrast so text remains readable.
- Keep file sizes compressed; transparent WebP can be used later if browser support and naming are updated.
